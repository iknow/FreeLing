Straight mode

When port-number is ommited, a stand-alone analyzer will be launched. It will load the configuration, read input from stdin, write results to stdout, and exit.
E.g.: 

 analyze -f en.cfg  <myinput  >myoutput

When the input file ends, the analyzer will stop and it will have to be reloaded again to process a new file.
